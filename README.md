# Reminder bot 2023

Простой telegram-bot напоминаний
- Backend, rest api.
- Стэк - kotlin, ktor, postgresql.

## Сценарий пользователя
1. Пользователь открывает телеграм бот.
2. Пользователь вводит напоминание в формате '1h 30m проверить почту', где в начале вводятся часы или минуты, а потом текст.
Если пользователь введет сообщение без указания таймера, то напоминание не будет обработано и сохранено.
3. Через 1час 30мин бот напоминает пользователю 'проверить почту'.

## Сценарий бота
1. Система с определенной периодичностью проверяет для каких напоминаний пришло время (задается в настройках).
2. Система посылает наступившие напоминания пользователю, который его создал.
Если нет напоминаний, то ничего не направляется.
3. Система помечает отправленное напоминание как "отработанное", чтобы не направлять его повторно.

## Дополнительные возможности (функции) пользователя:
1. Посмотреть все напоминания командой "all".
2. Удалить полученное напоминание ответом на него в виде команды "delete".
